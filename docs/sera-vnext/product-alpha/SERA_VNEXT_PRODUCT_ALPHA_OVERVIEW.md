# SERA vNext Product Alpha Overview

Date: 2026-06-06
Status: INTERNAL

## Product Status

```text
PRODUCT_ALPHA_CANDIDATE_ONLY_INTERNAL_READY
```

## Scope

Product Alpha allows an enterprise admin to submit event text and receive:

1. extracted facts;
2. summarized timeline;
3. non-final candidate escape window;
4. non-final reasoning lanes for perception, objective, and action;
5. warnings and uncertainties.

## Out Of Scope

1. final classification;
2. persistence;
3. export;
4. event mutation;
5. public access.
