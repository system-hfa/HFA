- During approach management, thrust lever and mode selections produced A/T HOLD state with no active automatic airspeed control.
- Crew did not detect mode transition in time, while vertical profile and airspeed diverged.
- PAPI and airspeed cues indicated unstable approach before 500 ft and continued degradation below glidepath.
- Low airspeed/low path awareness emerged late; go-around was initiated below effective recovery margin.
