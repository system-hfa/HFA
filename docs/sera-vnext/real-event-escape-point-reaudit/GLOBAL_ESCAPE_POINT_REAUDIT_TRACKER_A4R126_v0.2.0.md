## Reading Rules
- `escapePointWhenStatement` is mandatory.
- `ESCAPE_POINT_WHEN_STATEMENT_UNRESOLVED` means the event is not validated by the Hendy gate.
- `escapePointWhenStatementPreviouslyExplicit` records whether the mandatory "Quando..." field already existed before A4R126; all rows are `NO` because the field was introduced by this gate.
- `BLOCKED_ESCAPE_POINT_WHEN_STATEMENT_UNRESOLVED` is a formal when-statement gate block for every row whose statement is unresolved. - Prior P/O/A values are historical/draft statuses only. This tracker does not rebuild or correct them.
- `treeTraceRisk=HELPER_TRACE_ONLY` includes A4R80 questionPath backfills and other helper prompts.
- `treeTraceRisk=CANONICAL_TRACE_CONFIRMED` means exact canonical-question text is present in the dependency artifact, not that release is authorized.
