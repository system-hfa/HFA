## 5. Criterios de aceitacao

- relatorio oficial com locator rastreavel;
- cronologia minima para escape point;
- ator direto minimamente atribuivel;
- evidencia no pre-escape/escape;
- valor explicito para gap metodologico.

## 6. Criterios de rejeicao imediata

- somente noticia/jornal;
- sem relatorio oficial;
- sem cronologia;
- sem ator direto;
- ambiguidade excessiva;
- apenas conclusao causal sem fatos;
- utilidade apenas por severidade/outcome;
- relatorio inacessivel.
